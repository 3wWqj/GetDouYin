# 开发环境和开发工具
- JDK 1.8 版本
- IntelliJ IDEA 2018版本
- Hutool4.3.1版本（一个java工具类框架）


# 安装包
安装包 目录下有打包好的可运行文件
- 如果你懂Java，本机用jdk环境，运行`java -jar GetDouYin-1.0.jar`即可
- 如果你什么都不懂，我打包了一个windows版的 ，解压 douyin.zip 双击start.bat即可运行
- mac和linux版本暂时不弄了，需要的可以qq群下载 QQ群：317896269 代码狗
